# PRF Constructor - Визуальный конструктор примитивно-рекурсивных функций

## Описание

Интерактивное desktop-приложение на Python с графическим интерфейсом для построения, визуализации и вычисления примитивно-рекурсивных функций.

## Установка

### Требования
- Python 3.8 или выше
- Tkinter (обычно входит в стандартную поставку Python)
- SQLite3 (входит в стандартную поставку Python)
- Graphviz (опционально, для визуализации деревьев)

### Установка зависимостей

```bash
pip install -r requirements.txt
```

Для установки Graphviz:
- **Windows**: Скачайте с [официального сайта](https://graphviz.org/download/) или используйте `choco install graphviz`
- **macOS**: `brew install graphviz`
- **Linux**: `sudo apt-get install graphviz` (Ubuntu/Debian) или `sudo yum install graphviz` (CentOS/RHEL)

## Запуск

```bash
python main.py
```

## Быстрый старт

### 1. Загрузка предопределенных функций

В панели инструментов нажмите на кнопки с примерами:
- **add(x,y)** - сложение
- **mult(x,y)** - умножение
- **fact(x)** - факториал

### 2. Вычисление функций

1. Загрузите функцию (например, нажмите "add(x,y)")
2. В поле "Аргументы" введите значения через запятую: `5, 3`
3. Нажмите "Вычислить" или Enter
4. Результат отобразится в панели "Результат"

**Примеры:**
- `add(5, 3)` → `8`
- `mult(4, 5)` → `20`
- `fact(5)` → `120`

### 3. Сохранение и загрузка

- **Сохранить**: Меню Файл → Сохранить функцию (Ctrl+S)
- **Загрузить**: Меню Файл → Загрузить функцию (Ctrl+O)

### 4. Горячие клавиши

- **Ctrl+N** - Новая функция
- **Ctrl+S** - Сохранить
- **Ctrl+O** - Загрузить
- **F5** - Вычислить
- **Enter** - Вычислить (в поле аргументов)
