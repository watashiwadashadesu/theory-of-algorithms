# Почтовый Калькулятор

## Описание
Консольное приложение для расчета стоимости почтовых отправлений (письма, бандероли, посылки) в зависимости от расстояния, веса, объема и типа доставки.

## Функциональность
- Расчет стоимости для трех типов отправлений: Письмо, Бандероль, Посылка
- Учет различных параметров: расстояние, вес, объем, тип доставки, дополнительные характеристики
- Сохранение результатов в отчет формата Excel
- Модульная структура с использованием ООП

## Инструкции по запуску

2. Установите зависимости:
