# Менеджер книжного клуба

## Описание
Десктопное приложение для учета книг в книжном клубе. Позволяет вести каталог книг, отслеживать их статус, назначать читателей и выставлять рейтинги.

## Функциональность
- **CRUD для книг**: добавление, просмотр, редактирование, удаление
- **Визуализация**: таблица книг и круговая диаграмма по статусам
- **Логирование**: все действия сохраняются в лог-файл
- **Валидация**: проверка корректности вводимых данных
- **Статистика**: автоматический расчет статистики по книгам
- **Поиск и фильтрация** (в будущих версиях)

## Требования
- Python 3.8+
- PySide6
